# Educational Equipment Borrowing Module
A specialized mobile application module developed for managing inventory and tracking borrowed items like iPads and notebooks.

## 📱 Tech Stack
- **Frontend:** Flutter (Dart)
- **State Management:** Provider / Bloc (ตามที่คุณใช้)
- **Backend Integration:** Node.js API

## 🧩 My Contribution (Collaborative Project)
This repository contains the **Core Inventory & Borrowing Logic** module, which I independently developed as part of a larger team project.
- **Dynamic UI:** Developed responsive interfaces for equipment listing and status tracking.
- **Module Integration:** Designed the logic to interface with the main authentication system and backend APIs.
- **User Experience:** Implemented real-time status updates for item availability.

## 💡 Engineering Highlight
Focused on **Modular Design** to ensure seamless integration with the larger system. Prioritized code maintainability and clear API communication protocols.
